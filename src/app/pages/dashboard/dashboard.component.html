Medus Labs
Medus Labs allows you to quickly configure a number of member AWS accounts via the use of Cloud Formation Scripts.

The purpose of a lab is to provide a student with the minimum access required to complete the lesson plan and to be able to work in complete isolation from other students. When un-deploying we do all the hard work for you - removing the lab but also inspecting the students entire account and deleting any resources.
